# snowpacktest
Tried snowpack with reactjs and typescript

```zsh
$ cd snowpacktest

# install packages
$ yarn

# run devServer
$ yarn dev

# build
$ yarn build
```

You can use `npm` instead of `yarn`

```zsh
$ cd snowpacktest

# install packages
$ npm install

# run devServer
$ npm run dev

# build
$ npm run build
```
